# Sample Measurement Concept ID

## 1<sup>st</sup> release

* We plan to release sequentially sample measurement concept ID, which Samsung Medical Center and Ajou University School of Medicine in Korea worked together.
* We release the 66 laboratory measurements firstly.
* The OMOP concept IDs to be released are as follows.
	1. concept id
	2. concept name
	3. unit concept id
	4. operator concept id
	5. value as number (example)
	6. value as concept id
	7. LOINC UCUM units example
